  • element.querySelector(selector)

    ... современный стандарт для поиска элементов. Позволяют найти элемент или группу элементов по CSS-селектору любой сложности. Используется если необходимо найти только один, чаще всего уникальный элемент. Возвращает первый найденный элемент внутри element, соответствующий CSS-селектору selector. Если ничего не найдено, вернет null.
    Синтаксис:
    element.querySelector(selector);
  • element.querySelectorAll(selector)

    ... используется если необходимо найти коллекцию элементов, то есть получить массив ссылок на элементы с одинаковым селектором. Например, все элементы списка с классом menu-item. Возвращает псевдомассив всех элементов внутри element, удовлетворяющих CSS-селектору selector. Если ничего не найдено, вернет пустой массив.
    Синтаксис:
    element.querySelector(selector);
